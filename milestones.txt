MR NUTS FLOW MILESTONES
=====================

PROTOTYPE PHASE
--------------
1. Basic Squirrel Setup
   - Create squirrel entity with basic sprite
   - Add gravity component
   - Implement vertical falling

2. Basic State System
   - Implement state machine for squirrel (open/closed arms)
   - Add basic state switching with spacebar

3. Basic Movement
   - Implement rotation controls for open arms state
   - Add speed limits for each state
   - Implement acceleration in closed arms state

4. Basic Cloud Setup
   - Create cloud entities (white and black)
   - Add basic collision detection
   - Implement simple cloud placement

5. Basic Gameplay Loop
   - Add start position (helicopter drop)
   - Add simple end condition (reach bottom)
   - Implement basic timer

REFINEMENT PHASE

   - Add wiggle state implementation

   - Polish state transitions

2. Cloud System Enhancement
   - Implement different cloud sizes

   - Fine-tune cloud placement for better gameplay
   - Add cloud visual variations

3. Visual Polish
   - Add proper squirrel animations
   - Implement camera follow behavior

4. Game Flow
   - Add proper game start/end screens

5. Audio and Feedback
   - Add sound effects for state changes
